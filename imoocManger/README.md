#### 教程说明

 这里面是两个文件夹  
1. 一个是example是react的一些简单的小例子，可以根据官网和本文件进行演练熟悉 react的一些基本的知识
2. 另外一个是一个整体的项目 引入antUI组件 做的一个招聘求职的app 下面是APP文件的使用方式

#### 使用说明

1. cd imooc             //进入整体项目目录
2. npm install          //安装依赖
3. npm run server       //启动后台服务 端口9093
4. npm run start        //启动项目查看 端口3000
